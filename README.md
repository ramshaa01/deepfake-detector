# AI-Generated Face Detector

## Problem Statement
With the proliferation of generative AI, synthetic faces produced by models like StyleGAN2 have become indistinguishable from real photographs to the human eye. This creates serious risks for platforms handling user-generated content — fake profile creation, synthetic identity fraud, and AI-assisted impersonation at scale.

This project builds a binary classifier that detects whether a face image is a **genuine photograph of a real person** or an **entirely AI-generated synthetic face**, with interpretable, quantified confidence via Grad-CAM heatmaps and frequency-domain analysis.

> **Scope note:** This system detects AI-synthesised faces (GAN-generated), not video-based facial reenactment or manipulation techniques (e.g. FaceForensics++ Deepfakes / Face2Face / FaceSwap / NeuralTextures). Those require a different dataset and different failure modes. The distinction matters: GAN-synthesised fakes often have characteristic frequency-domain artefacts (spectral peaks from GAN upsampling), while video-based deepfakes tend to have blending boundary artefacts.

## Real-World Relevance
- **Fake profile detection** on social media / dating platforms
- **KYC / identity verification** pipelines needing synthetic face screening
- **Content moderation** for UGC platforms with synthetic persona risks
- **Research baseline** for understanding CNN-based GAN artefact detection

## Tech Stack
- **ML Core**: PyTorch, EfficientNet-B0 (transfer learning), OpenCV, MTCNN (face extraction), Grad-CAM (interpretability), FFT-based frequency features
  - *Python 3.11 required — needed for MTCNN/TensorFlow compatibility*
- **Backend**: FastAPI
- **Frontend**: React, Tailwind CSS, Recharts
- **Deployment**: HuggingFace Spaces / Render (Backend), Vercel (Frontend)

## Dataset

**Source:** [Kaggle — 140k Real and Fake Faces](https://www.kaggle.com/datasets/xhlulu/140k-real-and-fake-faces) (xhlulu)

| Class | Source | Nature |
|---|---|---|
| **Real (label=0)** | NVIDIA FFHQ (Flickr Faces HQ) | Genuine photographs of real people, 1024×1024 |
| **Fake (label=1)** | "1 Million Fake Faces" / StyleGAN2 | Entirely AI-generated synthetic portraits |

**What "fake" means here:** Images generated by StyleGAN2 — a GAN that synthesises photorealistic faces from scratch. There is no real person being manipulated; the face never existed. This is distinct from deepfake video manipulation where a real person's likeness is transplanted onto another video.

**Per-manipulation-type breakdown:** Not applicable. This dataset uses a single synthetic generation method (StyleGAN2), not multiple manipulation techniques. There is no Deepfakes / Face2Face / FaceSwap / NeuralTextures breakdown. Metrics are reported as binary real vs. synthetic accuracy only.

**Extracted subset used for training:**
- 2,000 balanced face crops (1,000 real, 1,000 fake) via MTCNN — 100% extraction success rate
- Split: 1,400 train / 300 val / 300 test — stratified, 50/50 class balance in every split
- Label convention: `0` = real photograph, `1` = AI-generated synthetic face

## Setup

```bash
# Python 3.11 required
py -3.11 -m venv venv
.\venv\Scripts\activate

pip install -r requirements.txt

# Download dataset (requires Kaggle API credentials in ~/.kaggle/kaggle.json)
python data/download_dataset.py --source kaggle --subset small

# Run face extraction (balanced 1000 real + 1000 fake)
python data/batch_extract.py

# Generate train/val/test splits
python data/make_splits.py

# Verify DataLoaders
python data/verify_dataloaders.py
```
