# Day 7: Official Head-Only Baseline Metrics

These metrics represent the performance of the **EfficientNet-B0 (Head-Only)** model on the isolated test set. This is the official baseline to beat during full fine-tuning (Day 8+).

## Metrics (Test Set)
- **Accuracy:** 0.7567
- **Precision:** 0.7516
- **Recall:** 0.7667
- **F1 Score:** 0.7591
- **ROC-AUC:** 0.8249

## Performance
- **Avg Inference Time:** 142.10 ms per image (on cpu)
